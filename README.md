# Primeleads - Modern Digital Agency Website

Primeleads is a modern, responsive digital agency website template built using HTML, CSS, and JavaScript. It is designed for agencies, startups, and freelancers who want a clean and professional online presence.

---

## 🚀 Features

- Fully responsive design (mobile, tablet, desktop)
- Modern UI with smooth animations
- Glassmorphism About section
- Services section with hover effects
- Portfolio grid layout
- Testimonials section
- Clients showcase
- Working contact form (EmailJS integration)
- Sticky navigation bar
- Clean and reusable code structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla JS)
- EmailJS (for contact form)
